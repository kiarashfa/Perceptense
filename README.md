# Have a Sense About Everything
**Perceptense** (from *perception* + *sense*) is a free, self-directed course made of bite-sized interactive modules.

It helps you build strong intuition for numbers, scale, science, culture, and the everyday systems that shape our world
Each module focuses on developing real intuition rather than rote memorization.
No sign-up. No ads. Just clear, visual, and practical knowledge.
